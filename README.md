#Project Title
E-Commerce Sales & Performance Dashboard (Power BI)

#Short Project Description

This project is an interactive Power BI dashboard built to analyze e-commerce sales and business performance. It converts raw online sales data into meaningful insights such as revenue trends, order volume, customer behavior, and product performance. The dashboard is designed to support data-driven decision-making through clear KPIs, dynamic filters, and business-oriented visualizations.

#Tech Stack & Tools

Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Measures & KPIs)
CSV / Excel Dataset
Dashboard Features

#Interactive KPI cards for sales, revenue, and order count

Dynamic slicers for time, category, and product-level analysis
Sales trend analysis using line and bar charts
Product and category-wise performance comparison
Clean and intuitive dashboard layout for quick insights
Optimized data model for faster report performance


Data Source

Online e-commerce dataset (publicly available sample data)
